### 9.2 优先级说明

| 优先级 | 定义 | 说明 |
|--------|------|------|
| P0 | 关键/阻塞 | 必须优先完成，否则阻塞其他任务 |
| P1 | 高 | 核心功能，应尽快完成 |
| P2 | 中 | 重要功能，按计划完成 |
| P3 | 低 | 可选功能，时间允许时完成 |

### 9.3 工时估算方法

使用PERT（Program Evaluation and Review Technique）方法：

```
期望工时 = (乐观 + 4×最可能 + 悲观) / 6
```

**示例**:
- 乐观: 2h
- 最可能: 3h
- 悲观: 4h
- 期望: (2 + 4×3 + 4) / 6 = 3h
