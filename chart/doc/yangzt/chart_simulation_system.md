# 海图导航预警模拟系统需求说明书

## 一、系统概述

### 1.1 系统定位

海图导航预警模拟系统是海图导航平台的专业辅助模块，提供航行模拟、预警演练、培训考核等功能。系统支持模拟模式和实际工作模式切换，允许用户在虚拟环境中预演航线、测试预警响应，提升航行安全水平。

### 1.2 系统目标

| 目标 | 具体指标 |
|-----|---------|
| 模拟真实性 | 模拟环境与实际航行高度一致 |
| 预警演练 | 支持各类预警场景模拟演练 |
| 培训考核 | 支持船员培训和考核评估 |
| 数据复用 | 历史航行数据可用于模拟 |

### 1.3 系统模式

| 模式 | 说明 | 适用场景 |
|-----|-----|---------|
| 实际工作模式 | 真实导航和预警 | 实际航行 |
| 模拟模式 | 虚拟航行模拟 | 培训演练、航线预演 |
| 回放模式 | 历史轨迹回放 | 事故分析、经验总结 |

### 2.1 模拟环境配置

### 3.2 预警响应评估

#### 3.2.1 评估指标

| 指标 | 说明 | 权重 |
|-----|-----|-----|
| 响应时间 | 从预警到响应的时间 | 30% |
| 决策正确性 | 响应措施是否正确 | 40% |
| 操作准确性 | 操作执行的准确性 | 20% |
| 结果安全性 | 最终结果是否安全 | 10% |

#### 3.2.2 评分标准

| 分数区间 | 等级 | 说明 |
|---------|-----|-----|
| 90-100 | 优秀 | 响应迅速、决策正确、操作准确 |
| 80-89 | 良好 | 响应及时、决策基本正确 |
| 70-79 | 中等 | 响应较慢、决策有偏差 |
| 60-69 | 及格 | 响应迟缓、需要改进 |
| <60 | 不及格 | 响应错误、需要重新培训 |
