# 海图预警系统需求说明书

## 一、系统概述

### 1.1 系统定位

海图预警系统是海图导航平台的安全保障模块，负责监测航行风险、推送预警信息、提供应急响应支持。系统通过整合多源数据，实现水深突变、恶劣天气、碰撞风险、航道关闭等多种预警类型的实时监测和推送。

### 1.2 系统目标

| 目标 | 具体指标 |
|-----|---------|
| 预警响应 | 提前30分钟预警（高级别），至少提前30秒 |
| 推送方式 | 应用内通知、短信（VIP）、声音警报 |
| 预警类型 | 水深突变、恶劣天气、碰撞风险、航道关闭等 |
| 阈值配置 | 系统预设+用户自定义 |
| 可靠性 | 冗余设计，保证预警不遗漏 |

### 1.3 预警级别定义

| 级别 | 名称 | 颜色 | 说明 |
|-----|-----|-----|-----|
| 一级 | 特别严重 | 红色 | 立即采取行动，生命安全威胁 |
| 二级 | 严重 | 橙色 | 尽快采取行动，重大风险 |
| 三级 | 较重 | 黄色 | 注意观察，潜在风险 |
| 四级 | 一般 | 蓝色 | 提示信息，轻微风险 |

## 二、预警类型详述

### 2.1 水深预警

#### 2.1.1 预警触发条件

| 预警级别 | 触发条件 | 响应时间 |
|---------|---------|---------|
| 一级（红色） | UKC < 0.3m 或水深 < 吃水 | 立即 |
| 二级（橙色） | UKC < 0.5m | 提前30min |
| 三级（黄色） | UKC < 1.0m | 提前30min |
| 四级（蓝色） | UKC < 1.5m | 提前30min |

#### 2.1.2 水深数据来源

| 数据源 | 更新频率 | 精度 | 适用场景 |
|-------|---------|-----|---------|
| 海图水深 | 静态 | 0.1m | 基础参考 |
| 实时测深 | 实时 | 0.01m | VIP船舶 |
| 潮汐修正 | 每小时 | 0.1m | 潮汐水域 |
| 航道通告 | 按需 | - | 临时变化 |

#### 2.1.3 水深预警算法

```
有效水深 = 海图水深 + 潮高 - 测量误差
UKC = 有效水深 - 船舶吃水 - 航行下沉量
```

| 参数 | 说明 |
|-----|-----|
| 海图水深 | 海图标注的水深值（基准面以下） |
| 潮高 | 当前潮位与基准面的差值 |
| 测量误差 | 水深测量不确定性 |
| 船舶吃水 | 船舶当前吃水深度 |
| 航行下沉量 | 航行中船体下沉量（Squat） |

#### 2.1.4 航行下沉量计算

**Barrass公式：**

```
Squat = Cb × V² / 100 (米)
```

其中：
- Cb = 方形系数（0.7-0.9）
- V = 航速（节）
- 浅水修正系数 = 1 + (15 × 水深/吃水 - 1)⁻¹

### 2.3 碰撞风险预警

#### 2.3.1 预警触发条件

| 预警级别 | CPA | TCPA | 说明 |
|---------|-----|------|-----|
| 一级（红色） | <0.5海里 | <6分钟 | 紧急碰撞风险 |
| 二级（橙色） | 0.5-1海里 | 6-15分钟 | 较高碰撞风险 |
| 三级（黄色） | 1-2海里 | 15-30分钟 | 潜在碰撞风险 |

#### 2.3.2 CPA/TCPA计算

**CPA（最近会遇距离）计算：**

```
CPA = |ΔR × sin(ΔC - ΔB)|
```

其中：
- ΔR = 两船距离
- ΔC = 相对航向
- ΔB = 目标方位

**TCPA（到达最近会遇点时间）计算：**

```
TCPA = ΔR × cos(ΔC - ΔB) / 相对速度
```

#### 2.3.3 AIS目标数据处理

| 处理步骤 | 说明 |
|---------|-----|
| 数据接收 | 接收AIS消息，解析目标信息 |
| 位置预测 | 根据航向航速预测目标位置 |
| 风险评估 | 计算与所有目标的CPA/TCPA |
| 风险排序 | 按风险等级排序目标 |
| 预警触发 | 超过阈值触发预警 |


### 3.1 推送方式

| 推送方式 | 适用场景 | 用户类型 |

## 五、预警响应流程

### 5.1 预警处理流程

```
数据采集 → 风险评估 → 预警判定 → 预警生成 → 推送分发 → 用户确认 → 预警解除
```

### 5.2 预警状态机

| 状态 | 说明 | 后续状态 |
|-----|-----|---------|
| 待处理 | 预警已生成，等待推送 | 已推送 |
| 已推送 | 预警已推送给用户 | 已确认/已解除 |
| 已确认 | 用户已确认预警 | 已解除 |
| 已解除 | 预警条件已消除 | 结束 |
| 已过期 | 预警超时未处理 | 结束 |

### 5.3 应急响应建议

| 预警类型 | 响应建议 |
|---------|---------|
| 水深预警 | 减速、改变航线、等待潮水 |
| 气象预警 | 就近避风、停止作业、加强瞭望 |
| 碰撞预警 | 调整航向航速、加强瞭望、联系对方 |
| 航道关闭 | 改变航线、等待开放、联系VTS |
