# UKC计算详述

## 一、概述

### 1.1 UKC定义

UKC（Under Keel Clearance，富余水深）是指船舶龙骨最低点至海底或航道底部的垂直距离，是衡量船舶航行安全的重要指标。

```
UKC = 实际水深 - 船舶吃水
```

### 1.2 UKC的重要性

| 重要性 | 说明 |
|-------|-----|
| 航行安全 | 防止船舶搁浅事故 |
| 航道规划 | 确定可通航水域范围 |
| 载货优化 | 最大化利用船舶载重能力 |
| 调度决策 | 辅助潮汐窗口期决策 |

### 1.3 UKC组成要素

| 要素 | 说明 | 影响因素 |
|-----|-----|---------|
| 静态UKC | 基础安全余量 | 船舶类型、航道条件 |
| 动态UKC | 航行中下沉量 | 航速、水深、船型 |
| 水位修正 | 潮汐水位变化 | 潮汐、径流 |
| 测量误差 | 水深测量不确定性 | 测量精度、海底地形 |

## 二、UKC计算公式

### 2.1 基本计算公式

```
UKC = H - T - S - E
```

其中：
- H：实际水深（海图水深 + 潮高）
- T：船舶吃水
- S：航行下沉量（Squat）
- E：测量误差和安全余量

### 2.2 完整计算公式

```
UKC = (H_chart + H_tide - H_datum) - T_static - T_dynamic - S - E - W
```

| 参数 | 说明 |
|-----|-----|
| H_chart | 海图水深 |
| H_tide | 潮高 |
| H_datum | 基准面差值 |
| T_static | 静态吃水 |
| T_dynamic | 动态吃水变化 |
| S | 航行下沉量 |
| E | 测量误差 |
| W | 横摇/纵摇增加吃水 |

## 三、航行下沉量计算

### 3.1 下沉量成因

| 成因 | 说明 |
|-----|-----|
| 伯努利效应 | 船体周围流速增加，压力降低 |
| 回流效应 | 船体排开水流形成回流 |
| 浅水效应 | 浅水中流动受限，下沉加剧 |
| 船型影响 | 船型系数影响下沉程度 |

### 3.2 Barrass公式

**适用于开阔水域：**

```
Squat = Cb × V² / 100 (米)
```

其中：
- Cb：方形系数（Block Coefficient）
- V：航速（节）

**适用于受限水域：**

```
Squat = Cb × V² / 100 × K
```

其中：
- K：受限系数 = 1 + (15 × H/T - 1)⁻¹
- H：水深
- T：吃水

### 3.3 Tuck公式

**适用于浅水航道：**

```
Squat = Cb × V² / (g × T) × F(h)
```

其中：
- g：重力加速度（9.81 m/s²）
- F(h)：水深函数

### 3.4 IMO推荐公式

```
Squat = 2.4 × Cb × (V/10)² × (T/H)^(1/3)
```

### 3.5 下沉量计算示例

**船舶参数：**
- 船长：180m
- 船宽：28m
- 吃水：10m
- 方形系数Cb：0.75
- 航速：12节
- 水深：15m

**计算过程：**

```
受限系数 K = 1 + (15 × 15/10 - 1)⁻¹ = 1 + 0.071 = 1.071
Squat = 0.75 × 12² / 100 × 1.071 = 1.08 × 1.071 = 1.16m
```

### 3.6 下沉量对照表

| 航速(节) | Cb=0.6 | Cb=0.7 | Cb=0.8 | Cb=0.9 |
|---------|--------|--------|--------|--------|
| 6 | 0.22m | 0.25m | 0.29m | 0.32m |
| 8 | 0.38m | 0.45m | 0.51m | 0.58m |
| 10 | 0.60m | 0.70m | 0.80m | 0.90m |
| 12 | 0.86m | 1.01m | 1.15m | 1.30m |
| 14 | 1.18m | 1.37m | 1.57m | 1.76m |
| 16 | 1.54m | 1.79m | 2.05m | 2.30m |

## 四、动态吃水计算

### 4.1 纵倾变化

| 因素 | 影响 | 计算方法 |
|-----|-----|---------|
| 油水消耗 | 纵倾变化 | 按消耗量计算力矩 |
| 货物移动 | 纵倾变化 | 按移动距离计算 |
| 下沉差异 | 首尾下沉不同 | 按船型系数计算 |

### 4.2 横摇增加吃水

```
ΔT_roll = B/2 × sin(θ)
```

其中：
- B：船宽
- θ：横摇角度

**横摇角度参考值：**

| 海况 | 横摇角度 |
|-----|---------|
| 平静 | 0-5° |
| 轻浪 | 5-10° |
| 中浪 | 10-15° |
| 大浪 | 15-20° |

### 4.3 纵摇增加吃水

```
ΔT_pitch = L/2 × sin(φ)
```

其中：
- L：船长
- φ：纵摇角度

## 五、水深计算

### 5.1 海图水深

| 项目 | 说明 |
|-----|-----|
| 基准面 | 理论最低潮面（LAT）或海图基准面（CD） |
| 水深值 | 基准面以下的水深 |
- 精度 | 通常为0.1m |

### 5.2 潮高计算

#### 5.2.1 潮汐预报法

```
潮高 = 平均海面 + 潮差 × f(t)
```

其中f(t)为潮汐调和函数。

#### 5.2.2 高低潮插值法

```
潮高 = H_low + (H_high - H_low) × cos²(π × t/T)
```

其中：
- H_high：高潮高
- H_low：低潮高
- T：高低潮时间间隔
- t：距低潮时间

#### 5.2.3 潮高计算示例

**已知条件：**
- 低潮时间：06:30，潮高：1.2m
- 高潮时间：12:45，潮高：3.8m
- 当前时间：09:00

**计算过程：**
```
T = 12:45 - 06:30 = 6小时15分 = 6.25小时
t = 09:00 - 06:30 = 2.5小时
潮高 = 1.2 + (3.8 - 1.2) × cos²(π × 2.5/6.25)
     = 1.2 + 2.6 × cos²(0.4π)
     = 1.2 + 2.6 × 0.345
     = 2.10m
```

### 5.3 实际水深计算

```
实际水深 = 海图水深 + 潮高
```

**注意事项：**
- 确认海图基准面与潮汐基准面一致
- 注意基准面差值修正
- 考虑气压变化对水位的影响

## 八、UKC计算代码实现

### 8.1 Python实现

```python
import math
from dataclasses import dataclass
from datetime import datetime
from typing import Optional

@dataclass
class ShipParameters:
    length: float
    beam: float
    draft: float
    block_coefficient: float
    ukc_standard: float

@dataclass
class UKCResult:
    actual_depth: float
    squat: float
    dynamic_draft: float
    ukc: float
    ukc_percentage: float
    status: str
    warning: Optional[str]

class UKCCalculator:
    def __init__(self, ship: ShipParameters):
        self.ship = ship
    
    def calculate_squat(self, speed: float, depth: float) -> float:
        """
        计算航行下沉量（Barrass公式）
        
        Args:
            speed: 航速（节）
            depth: 水深（米）
        
        Returns:
            下沉量（米）
        """
        h_t_ratio = depth / self.ship.draft
        if h_t_ratio > 1.5:
            K = 1.0
        else:
            K = 1.0 + 1.0 / (15.0 * h_t_ratio - 1.0)
        
        squat = self.ship.block_coefficient * (speed ** 2) / 100.0 * K
        return round(squat, 2)
    
    def calculate_roll_draft_increase(self, roll_angle: float) -> float:
        """
        计算横摇增加吃水
        
        Args:
            roll_angle: 横摇角度（度）
        
        Returns:
            增加吃水（米）
        """
        return round(self.ship.beam / 2.0 * math.sin(math.radians(roll_angle)), 2)
    
    def calculate_pitch_draft_increase(self, pitch_angle: float) -> float:
        """
        计算纵摇增加吃水
        
        Args:
            pitch_angle: 纵摇角度（度）
        
        Returns:
            增加吃水（米）
        """
        return round(self.ship.length / 2.0 * math.sin(math.radians(pitch_angle)), 2)
    
    def calculate_ukc(
        self,
        chart_depth: float,
        tide_height: float,
        speed: float,
        roll_angle: float = 0.0,
        pitch_angle: float = 0.0,
        measurement_error: float = 0.3
    ) -> UKCResult:
        """
        计算UKC
        
        Args:
            chart_depth: 海图水深（米）
            tide_height: 潮高（米）
            speed: 航速（节）
            roll_angle: 横摇角度（度）
            pitch_angle: 纵摇角度（度）
            measurement_error: 测量误差（米）
        
        Returns:
            UKC计算结果
        """
        actual_depth = chart_depth + tide_height
        
        squat = self.calculate_squat(speed, actual_depth)
        
        roll_increase = self.calculate_roll_draft_increase(roll_angle)
        pitch_increase = self.calculate_pitch_draft_increase(pitch_angle)
        
        dynamic_draft = self.ship.draft + squat + roll_increase + pitch_increase
        
        ukc = actual_depth - dynamic_draft - measurement_error
        
        ukc_percentage = (ukc / self.ship.draft) * 100.0
        
        if ukc < 0:
            status = "danger"
            warning = "UKC为负值，存在搁浅风险！"
        elif ukc < self.ship.ukc_standard * 0.5:
            status = "warning"
            warning = f"UKC低于安全标准{self.ship.ukc_standard}m的50%"
        elif ukc < self.ship.ukc_standard:
            status = "caution"
            warning = f"UKC低于安全标准{self.ship.ukc_standard}m"
        else:
            status = "safe"
            warning = None
        
        return UKCResult(
            actual_depth=round(actual_depth, 2),
            squat=squat,
            dynamic_draft=round(dynamic_draft, 2),
            ukc=round(ukc, 2),
            ukc_percentage=round(ukc_percentage, 1),
            status=status,
            warning=warning
        )

def main():
    ship = ShipParameters(
        length=180.0,
        beam=28.0,
        draft=10.0,
        block_coefficient=0.75,
        ukc_standard=1.0
    )
    
    calculator = UKCCalculator(ship)
    
    result = calculator.calculate_ukc(
        chart_depth=12.0,
        tide_height=2.5,
        speed=12.0,
        roll_angle=5.0,
        measurement_error=0.3
    )
    
    print(f"实际水深: {result.actual_depth}m")
    print(f"下沉量: {result.squat}m")
    print(f"动态吃水: {result.dynamic_draft}m")
    print(f"UKC: {result.ukc}m ({result.ukc_percentage}%)")
    print(f"状态: {result.status}")
    if result.warning:
        print(f"警告: {result.warning}")

if __name__ == "__main__":
    main()
```

### 8.2 Java实现

```java
public class UKCCalculator {
    private final ShipParameters ship;
    
    public UKCCalculator(ShipParameters ship) {
        this.ship = ship;
    }
    
    public double calculateSquat(double speed, double depth) {
        double hTRatio = depth / ship.getDraft();
        double K;
        if (hTRatio > 1.5) {
            K = 1.0;
        } else {
            K = 1.0 + 1.0 / (15.0 * hTRatio - 1.0);
        }
        return ship.getBlockCoefficient() * Math.pow(speed, 2) / 100.0 * K;
    }
    
    public UKCResult calculateUKC(double chartDepth, double tideHeight, 
                                   double speed, double rollAngle, 
                                   double pitchAngle, double measurementError) {
        double actualDepth = chartDepth + tideHeight;
        double squat = calculateSquat(speed, actualDepth);
        
        double rollIncrease = ship.getBeam() / 2.0 * Math.sin(Math.toRadians(rollAngle));
        double pitchIncrease = ship.getLength() / 2.0 * Math.sin(Math.toRadians(pitchAngle));
        
        double dynamicDraft = ship.getDraft() + squat + rollIncrease + pitchIncrease;
        double ukc = actualDepth - dynamicDraft - measurementError;
        double ukcPercentage = (ukc / ship.getDraft()) * 100.0;
        
        String status;
        String warning = null;
        if (ukc < 0) {
            status = "danger";
            warning = "UKC为负值，存在搁浅风险！";
        } else if (ukc < ship.getUkcStandard() * 0.5) {
            status = "warning";
            warning = "UKC低于安全标准50%";
        } else if (ukc < ship.getUkcStandard()) {
            status = "caution";
            warning = "UKC低于安全标准";
        } else {
            status = "safe";
        }
        
        return new UKCResult(actualDepth, squat, dynamicDraft, ukc, 
                            ukcPercentage, status, warning);
    }
}

## 九、UKC预警阈值设置

### 9.1 预警级别

| 级别 | UKC阈值 | 颜色 | 响应措施 |
|-----|--------|-----|---------|
| 安全 | UKC ≥ 标准 | 绿色 | 正常航行 |
| 注意 | 标准 > UKC ≥ 标准×0.7 | 黄色 | 加强瞭望 |
| 警告 | 标准×0.7 > UKC ≥ 标准×0.5 | 橙色 | 减速航行 |
| 危险 | 标准×0.5 > UKC ≥ 0 | 红色 | 立即采取措施 |
| 紧急 | UKC < 0 | 紫色 | 紧急避险 |

### 9.2 预警响应建议

| UKC状态 | 响应建议 |
|---------|---------|
| 注意 | 关注水深变化，准备减速 |
| 警告 | 立即减速，评估改道可能性 |
| 危险 | 大幅减速或停车，寻找深水区 |
| 紧急 | 立即停车，评估搁浅风险 |

## 十、UKC计算注意事项

### 10.1 数据准确性

| 注意项 | 说明 |
|-------|-----|
| 吃水数据 | 使用实际吃水，非设计吃水 |
| 水深数据 | 确认基准面一致 |
| 潮高数据 | 使用最近的潮汐站数据 |
| 航速数据 | 使用对地速度 |

### 10.2 计算时机

| 时机 | 说明 |
|-----|-----|
| 出航前 | 评估全程UKC |
| 进入浅水区前 | 重点计算 |
| 潮汐变化时 | 更新潮高数据 |
| 航速变化时 | 重新计算下沉量 |

### 10.3 特殊情况

| 情况 | 处理方法 |
|-----|---------|
| 水深数据缺失 | 保守估计，增加安全余量 |
| 潮汐数据缺失 | 使用最低潮位计算 |
| 海底地形复杂 | 增加测量误差估计 |
| 恶劣天气 | 增加横摇/纵摇修正 |

### 11.3 参数计算阶段

#### 11.3.1 航行下沉量计算流程

**计算步骤：**

| 步骤 | 计算内容 | 公式 | 输入参数 |
|------|---------|------|---------|
| 1 | 判断水域类型 | H/T比值 | 水深H、吃水T |
| 2 | 计算受限系数K | K = 1 + 1/(15×H/T - 1) | H/T比值 |
| 3 | 计算基础下沉量 | S_base = Cb × V² / 100 | 方形系数Cb、航速V |
| 4 | 计算实际下沉量 | S = S_base × K | S_base、K |

**详细计算逻辑：**

```
┌─────────────────────────────────────────────────────────────┐
│                    下沉量计算逻辑                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  输入:                                                      │
│    - 船舶吃水 T = 10.0m                                     │
│    - 方形系数 Cb = 0.75                                     │
│    - 航速 V = 12节                                          │
│    - 水深 H = 15.0m                                         │
│                                                             │
│  步骤1: 计算水深吃水比                                       │
│    H/T = 15.0 / 10.0 = 1.5                                  │
│                                                             │
│  步骤2: 判断水域类型                                         │
│    if H/T > 1.5:                                            │
│        水域类型 = "开阔水域"                                 │
│        K = 1.0                                              │
│    else:                                                    │
│        水域类型 = "受限水域"                                 │
│        K = 1 + 1/(15 × 1.5 - 1) = 1.048                     │
│                                                             │
│  步骤3: 计算基础下沉量                                       │
│    S_base = Cb × V² / 100                                   │
│           = 0.75 × 12² / 100                                │
│           = 1.08m                                           │
│                                                             │
│  步骤4: 计算实际下沉量                                       │
│    S = S_base × K                                           │
│      = 1.08 × 1.048                                         │
│      = 1.13m                                                │
│                                                             │

│                                                             │

#### 11.3.2 动态吃水计算流程

**动态吃水组成：**

```
动态吃水 = 静态吃水 + 下沉量 + 横摇增量 + 纵摇增量 + 其他修正
```

| 组成部分 | 计算公式 | 影响因素 |
|---------|---------|---------|
| 静态吃水 | T | 装载状态 |
| 下沉量 | S = Cb × V² / 100 × K | 航速、水深 |
| 横摇增量 | ΔT_roll = B/2 × sin(θ) | 船宽、横摇角 |
| 纵摇增量 | ΔT_pitch = L/2 × sin(φ) | 船长、纵摇角 |
| 波浪修正 | ΔT_wave | 波高 |
| 淡水修正 | ΔT_fresh = T × (ρ_sea/ρ_fresh - 1) | 水密度 |

**横摇角度估算：**

| 海况等级 | 风级 | 浪高 | 横摇角度估算 |
|---------|-----|-----|------------|
| 平静 | ≤3级 | ≤0.5m | 0-3° |
| 轻浪 | 4-5级 | 0.5-1.25m | 3-8° |
| 中浪 | 5-6级 | 1.25-2.5m | 8-15° |
| 大浪 | 6-7级 | 2.5-4.0m | 15-20° |
| 巨浪 | ≥8级 | ≥4.0m | >20° |

### 11.4 UKC计算阶段

#### 11.4.1 UKC核心计算公式

```
UKC = H_actual - T_dynamic - E
```

其中：
- H_actual = H_chart + H_tide（实际水深）
- T_dynamic = T_static + S + ΔT（动态吃水）
- E = 测量误差 + 安全余量

#### 11.4.3 UKC计算伪代码

```python
def calculate_ukc(ship_params, env_data, position):
    """
    UKC计算主函数
    
    Args:
        ship_params: 船舶参数（吃水、船型系数等）
        env_data: 环境数据（潮高、风浪等）
        position: 船位（经纬度）
    
    Returns:
        UKC计算结果
    """
    
    # 1. 获取海图水深
    chart_depth = get_chart_depth(position)
    
    # 2. 获取潮高
    tide_height = get_tide_height(position, current_time)
    
    # 3. 计算实际水深
    actual_depth = chart_depth + tide_height
    
    # 4. 获取船舶吃水
    static_draft = ship_params.draft
    
    # 5. 计算下沉量
    squat = calculate_squat(
        cb=ship_params.block_coefficient,
        speed=ship_params.speed,
        depth=actual_depth,
        draft=static_draft
    )
    
    # 6. 计算横摇增量
    roll_angle = estimate_roll_angle(env_data.wind, env_data.wave)
    roll_increase = ship_params.beam / 2 * sin(radians(roll_angle))
    
    # 7. 计算纵摇增量
    pitch_angle = estimate_pitch_angle(env_data.wave)
    pitch_increase = ship_params.length / 2 * sin(radians(pitch_angle))
    
    # 8. 计算动态吃水
    dynamic_draft = static_draft + squat + max(roll_increase, pitch_increase)
    
    # 9. 获取测量误差
    measurement_error = get_measurement_error(chart_depth)
    
    # 10. 计算UKC
    ukc = actual_depth - dynamic_draft - measurement_error
    
    # 11. 安全评估
    status = evaluate_safety(ukc, ship_params.ukc_standard)
    
    return {
        'actual_depth': actual_depth,
        'squat': squat,
        'dynamic_draft': dynamic_draft,
        'ukc': ukc,
        'ukc_percentage': ukc / static_draft * 100,
        'status': status
    }
```

**预警判定逻辑：**

```python
def evaluate_safety(ukc, ukc_standard):
    """
    安全评估函数
    
    Args:
        ukc: 计算得到的UKC值
        ukc_standard: UKC标准值
    
    Returns:
        (status, level, color, recommendation)
    """
    
    if ukc >= ukc_standard:
        return ("安全", 1, "绿色", "正常航行")
    elif ukc >= ukc_standard * 0.8:
        return ("注意", 2, "蓝色", "加强瞭望")
    elif ukc >= ukc_standard * 0.6:
        return ("警戒", 3, "黄色", "减速航行")
    elif ukc >= ukc_standard * 0.4:
        return ("警告", 4, "橙色", "大幅减速")
    elif ukc > 0:
        return ("危险", 5, "红色", "立即采取措施")
    else:
        return ("紧急", 6, "紫色", "紧急避险")
```
