# 航运业各流程中Excel使用详细说明书

## 一、概述

### 1.1 文档目的

本文档详细说明航运业各业务流程中Excel的使用方法，包括数据录入、计算公式、数据分析、报表生成等操作指南，帮助航运从业人员高效使用Excel完成日常工作。

### 1.2 适用范围

| 适用人员 | 适用场景 |
|---------|---------|
| 船员 | 航行记录、燃油消耗统计 |
| 船长 | 航行计划、货物配载 |
| 调度员 | 船舶调度、航次统计 |
| 财务人员 | 成本核算、费用统计 |
| 管理人员 | 数据分析、报表制作 |

## 二、航行计划Excel模板

### 2.1 航行计划表结构

| 列号 | 列名 | 数据类型 | 说明 |
|-----|-----|---------|-----|
| A | 航次编号 | 文本 | 格式：VYYYYMMDD-XXX |
| B | 船名 | 文本 | 船舶名称 |
| C | 出发港 | 文本 | 起始港口 |
| D | 目的港 | 文本 | 目的港口 |
| E | 出发时间 | 日期时间 | 格式：YYYY-MM-DD HH:MM |
| F | 预计到达时间 | 日期时间 | 格式：YYYY-MM-DD HH:MM |
| G | 航程(海里) | 数值 | 航行距离 |
| H | 平均航速(节) | 数值 | 预计平均航速 |
| I | 预计航行时间(小时) | 公式 | =G/H |
| J | 燃油消耗(吨) | 公式 | =I*燃油消耗率 |
| K | 货物类型 | 文本 | 货物描述 |
| L | 货物数量(吨) | 数值 | 货物重量 |
| M | 备注 | 文本 | 其他说明 |

### 2.2 关键公式

#### 2.2.1 航行时间计算

```excel
=ROUND((F2-E2)*24,1)
```

说明：计算两个时间点之间的小时数

#### 2.2.2 燃油消耗计算

```excel
=I2*VLOOKUP(B2,船舶参数表!A:C,3,FALSE)
```

说明：根据船名查找燃油消耗率并计算总消耗

#### 2.2.3 预计到达时间计算

```excel
=E2+G2/H2/24
```

说明：出发时间+航行时间（小时转换为天数）

### 2.3 条件格式设置

| 条件 | 格式 | 说明 |
|-----|-----|-----|
| 燃油消耗>50吨 | 红色填充 | 高油耗警示 |
| 航行时间>24小时 | 黄色填充 | 长航程提示 |
| 预计到达时间已过 | 灰色填充 | 已完成航次 |

### 3.1 货物配载表结构

| 列号 | 列名 | 数据类型 | 说明 |
|-----|-----|---------|-----|
| A | 舱室编号 | 文本 | 如：NO.1舱 |
| B | 货物名称 | 文本 | 货物描述 |
| C | 货物重量(吨) | 数值 | 货物重量 |
| D | 货物体积(m³) | 数值 | 货物体积 |
| E | 积载因数(m³/t) | 公式 | =D/C |
| F | 舱容利用率 | 公式 | =D/舱容 |
| G | 重心高度(m) | 数值 | 货物重心高度 |
| H | 纵向位置(m) | 数值 | 距船中距离 |
| I | 垂向力矩 | 公式 | =C*G |
| J | 纵向力矩 | 公式 | =C*H |

### 3.2 配载计算公式

#### 3.2.1 总重量计算

```excel
=SUM(C2:C20)
```

#### 3.2.2 总力矩计算

```excel
=SUM(I2:I20)
```

#### 3.2.3 合成重心高度计算

```excel
=SUMPRODUCT(C2:C20,G2:G20)/SUM(C2:C20)
```

#### 3.2.4 纵倾计算

```excel
=(SUMPRODUCT(C2:C20,H2:H20)/SUM(C2:C20)-LCF)/MTC*100
```

说明：LCF为漂浮中心纵向位置，MTC为每厘米纵倾力矩

### 3.3 稳性校核

| 校核项目 | 公式 | 要求 |
|---------|-----|-----|
| 初稳性高度GM | =KM-KG | GM≥0.15m |
| 最大静稳性力臂 | 查稳性横交曲线 | GZmax≥0.20m |
| 静稳性面积 | 数值积分 | 面积≥0.055m·rad |

## 四、燃油管理Excel模板

### 4.1 燃油消耗记录表

| 列号 | 列名 | 数据类型 | 说明 |
|-----|-----|---------|-----|
| A | 日期 | 日期 | YYYY-MM-DD |
| B | 时间 | 时间 | HH:MM |
| C | 主机转速(RPM) | 数值 | 主机转速 |
| D | 主机功率(kW) | 数值 | 主机功率 |
| E | 航速(节) | 数值 | 当前航速 |
| F | 主机油耗(L/h) | 数值 | 主机燃油消耗 |
| G | 辅机油耗(L/h) | 数值 | 辅机燃油消耗 |
| H | 锅炉油耗(L/h) | 数值 | 锅炉燃油消耗 |
| I | 总油耗(L/h) | 公式 | =F+G+H |
| J | 累计油耗(L) | 公式 | =上一行J+I*时间间隔 |
| K | 剩余油量(L) | 数值 | 油舱测量值 |

### 4.2 燃油消耗率计算

#### 4.2.1 日均油耗

```excel
=AVERAGEIF(A:A,TODAY(),I:I)*24
```

#### 4.2.2 航次总油耗

```excel
=SUMIFS(I:I,A:A,">="&航次开始日期,A:A,"<="&航次结束日期)*时间间隔
```

#### 4.2.3 单位运输油耗

```excel
=航次总油耗/航程/货物重量
```

### 4.3 燃油预警设置

| 预警类型 | 条件 | 公式 |
|---------|-----|-----|
| 低油量预警 | 剩余油量<安全储备 | =IF(K2<安全储备,"警告","正常") |
| 油耗异常 | 油耗率>标准值*1.2 | =IF(I2/E2>标准值*1.2,"异常","正常") |
| 油量差异 | 计算值与测量值差异>5% | =IF(ABS(J2-K2)/K2>0.05,"差异","正常") |

### 5.2 坐标转换公式

#### 5.2.1 度分格式转十进制度

```excel
=LEFT(C2,FIND("°",C2)-1)+MID(C2,FIND("°",C2)+1,FIND("'",C2)-FIND("°",C2)-1)/60
```

#### 5.2.2 十进制度转度分格式

```excel
=INT(ABS(D2))&"°"&ROUND((ABS(D2)-INT(ABS(D2)))*60,3)&"'"&IF(D2>=0,"E","W")
```

### 5.3 航程统计

#### 5.3.1 日航程计算

```excel
=SUMIFS(航速:航速,日期:日期,TODAY())*时间间隔
```

#### 5.3.2 累计航程计算

```excel
=SUMPRODUCT(航速范围*时间间隔)
```

### 9.2 宏录制应用

#### 9.2.1 录制日报生成宏

```
Sub 生成日报()
    ' 选择数据区域
    Range("A1:Z100").Select
    ' 复制到新工作表
    Selection.Copy
    Sheets.Add After:=Sheets(Sheets.Count)
    ActiveSheet.Paste
    ' 添加日期标题
    Range("A1").Value = "日报 - " & Format(Date, "YYYY-MM-DD")
    ' 打印设置
    ActiveSheet.PageSetup.Orientation = xlLandscape
    ActiveSheet.PrintPreview
End Sub
```

## 十一、Excel航线规划工作流程详述

### 11.1 航线规划流程概述

根据航线拟定程序的四阶段划分（评估、拟定、执行、监测），Excel主要应用于前两个阶段：

| 阶段 | 任务 | Excel角色 |
|------|------|----------|
| 评估阶段 | 收集所有相关航海信息 | 作为信息整合平台，二副将来自不同来源的数据手动录入工作表 |
| 拟定阶段 | 基于评估数据，详细规划航线各航路点的具体参数 | 二副使用电子表格手动填写每个航路点的详细参数 |

| 潮汐数据 | 潮汐表、潮流预报 | 潮汐表/APP | 手动录入潮高、潮时 |

#### 11.3.2 关键计算公式

**1. 航向计算（根据两航路点坐标）**

```excel
=DEGREES(ATAN2(
    SIN(RADIANS(经度2-经度1))*COS(RADIANS(纬度2)),
    COS(RADIANS(纬度1))*SIN(RADIANS(纬度2))-SIN(RADIANS(纬度1))*COS(RADIANS(纬度2))*COS(RADIANS(经度2-经度1))
))
```

**2. 航程计算（大圆距离）**

```excel
=ACOS(SIN(RADIANS(纬度1))*SIN(RADIANS(纬度2))+COS(RADIANS(纬度1))*COS(RADIANS(纬度2))*COS(RADIANS(经度2-经度1)))*60
```

**3. 转向角计算**

```excel
=ABS(航向2-航向1)
=IF(转向角>180,360-转向角,转向角)
```

**4. 转弯半径计算**

```excel
=船速^2/(10*转弯率)
```

**5. UKC计算（含下沉量）**

```excel
=实际水深-吃水-下沉量-安全余量
```

**下沉量计算（Barrass公式）：**

```excel
=方形系数*船速^2/100*受限系数
```
