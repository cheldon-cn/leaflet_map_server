# 近五年海图/水域图导航系统方向投标信息梳理

## 概述

导航系统方向近五年投标活动主要围绕船舶气象导航、智能导航系统、电子航道图导航、内河导航等展开。采购方包括交通运输部水运科学研究所、武汉船舶职业技术学院、宁波港建设开发有限公司等，供应商以大连海大智龙科技、武汉同为科技等为代表。该方向项目金额从数万到数百万不等，智能导航与电子海图融合是显著趋势。

## 趋势分析

导航系统方向呈现以下趋势：一是电子航道图建设加速，河北、四川、江苏等多省推进内河电子航道图测绘和系统升级，天津路航信息科技成为该领域新兴供应商；二是智能导航与电子海图深度融合，武汉船舶职业技术学院的"船舶智能导航与电子海图实训室"项目直接体现了这一方向；三是气象导航成为新增长点，交通运输部水运科学研究所采购船舶气象导航实验系统；四是智慧航道平台化发展，河南、江苏等省建设智慧航道综合监管平台。该方向市场容量估计在3000万-8000万元/年，以航道管理部门和交通系统为主要客户。
