# 近五年海图/水域图港口方向投标信息梳理

## 概述

港口方向近五年投标活动主要围绕智慧港口、港口信息化、港口数字化管理、港口监控系统等展开。采购方包括港口集团（江苏盐城港、广州港、山东港口等）、港口运营企业等，供应商以葛洲坝通信技术、中电鸿信、浪潮软件、交通运输部天津水运工程科学研究所等为代表。该方向项目金额跨度最大，从百万级到近亿元不等，是六个方向中单体项目金额最高的领域。

## 趋势分析

港口方向呈现以下趋势：一是投资规模巨大，江苏盐城港9950万元的智慧港口安全能源管控项目是所有方向中单体金额最大的项目；二是港口全面数字化，从港口设施管理（广州港数字化管理系统）到安全监控（嘉兴港监控一张图）到能源管控，形成全方位数字化格局；三是5G和机器人等新技术应用，周口临港开发区项目涵盖5G专网、巡检机器人、无人机等；四是港口监控持续投入，多家港口集团持续采购和升级视频监控系统；五是水运科研院所参与港口数字化建设，交通运输部天津水运工程科学研究所中标广州港数字化管理系统。该方向市场容量估计在2亿-5亿元/年，是六大方向中市场体量最大的领域，且大型项目集中在港口集团和港务企业。
