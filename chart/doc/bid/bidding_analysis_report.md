# 近五年海图/水域图投标信息综合分析报告

## 摘要

本报告梳理了2021-2026年海图、内河航道图、水域图在显示、预警、模拟、导航、渔业养殖、港口六大方向的投标/招标信息。研究发现，六大方向合计年市场容量约在4亿-10亿元区间，其中港口方向单体项目金额最大（最高近1亿元），渔业养殖方向增长最快（受海洋牧场政策驱动），显示与预警方向集中度最高（华海海图成为海事系统核心供应商）。海图S-101标准升级、智慧航道/港口建设、海洋牧场信息化是三大核心驱动力。

## 研究背景

随着国家"海洋强国""交通强国"战略推进，以及IHO S-101电子海图新标准的发布，海图及水域信息化领域迎来新一轮投资周期。同时，海洋牧场示范区建设、智慧港口和智慧航道发展规划等政策持续释放市场需求。本报告通过梳理近五年公开招标信息，为相关企业和研究机构提供市场参考。

## 各方向核心发现

### 显示系统方向

电子海图显示系统方向的投标活动集中在两个层面：一是海事系统的大型平台建设，交通运输部海事局连续两年大额采购（1988万元智慧海事综合管理系统+1735.8万元一体化建设项目），华海海图（北京）科技有限公司成为海事系统核心供应商；二是S-101标准升级推动的系统性更新，上海海图中心530.8万元的S-101电子海图生产系统建设工程是标志性项目，由东华软件智能科技中标。

从S-57向S-101标准迁移是该方向最显著的技术趋势。S-101是IHO新一代电子海图标准，支持更丰富的数据模型和更灵活的显示方式，海图中心需要建设全新的生产系统来适配。同时，ECDIS船载设备的批量更新也是稳定需求来源，国家能源集团航运公司8艘船16套ECDIS更新是典型案例。

### 预警系统方向

预警系统方向的项目可以分为三类：一是国家级预警平台建设，国家海洋环境预报中心554万元的新一代海洋预警报业务平台是代表性项目，涵盖海啸业务操作平台、预警操作平台、业务展示平台等完整体系；二是VTS系统升级，福建海事局879万元的VTS改造工程配套设备采购体现了交通管理系统向智能化演进；三是地方海事处安全监测系统，杭州麟云科技在四川多地中标水上安全监测项目。

新技术开始渗透预警领域，上海海事大学采购的水路交通韧性全息感知系统（69.9万元）和XX安全航行仿真分析系统项目代表了全息感知、仿真分析等新方法在预警中的应用趋势。

### 模拟仿真方向

模拟仿真方向以航海院校为主要采购方，项目金额相对较低（38-276万元），但项目数量多。最突出的趋势是国产化替代，浙江交通职业技术学院275.78万元采购"国产大型船舶操纵模拟器"，大连海事大学中标成为国产供应商代表。大连海大智龙科技在多所院校中标模拟器采购和升级项目，是另一个核心供应商。

航道通航仿真是该方向的新应用场景，山东港口青岛港采购航道通航仿真及操船模拟专题研究，将仿真技术从培训延伸到港口航道规划设计领域。军队需求也在增加，多个军队采购项目涉及航海模拟仿真系统。

### 导航系统方向

导航系统方向的核心趋势是电子航道图建设和智能导航与电子海图融合。河北、四川、江苏等多省推进内河电子航道图测绘和系统升级，天津路航信息科技中标河北省电子航道图测绘项目成为该领域新兴供应商。

气象导航是新增长点，交通运输部水运科学研究所采购船舶气象导航实验系统，大连海大智龙科技中标。河南省251.27万元的智慧航道系统融合项目代表了航道导航系统向平台化、综合化方向发展的趋势。

### 渔业养殖方向

渔业养殖方向是六大方向中增长最快的领域，核心驱动力是国家海洋牧场示范区建设政策。遂溪县605万元的海洋牧场智慧渔业监管决策平台是该方向最大项目。从辽宁到海南，沿海各省都在推进海洋牧场数字化监测和智慧渔业平台建设。

值得关注的是，内陆地区也开始出现智慧渔业项目，陕西合阳县智慧渔业信息化平台、湖北竹溪县智慧化设施渔业建设项目（EPC）等，说明渔业信息化需求已从沿海扩展到内陆。

### 港口方向

港口方向是六大方向中市场体量最大的领域，江苏盐城港9950万元的智慧港口安全能源管控项目是所有方向中单体金额最大的项目。港口数字化涵盖设施管理（广州港数字化管理系统）、安全监控（嘉兴港监控一张图）、能源管控等多个维度，形成全方位数字化格局。

5G、机器人、无人机等新技术开始在港口领域应用，周口临港开发区项目涵盖5G专网、巡检机器人、无人机等。交通运输部天津水运工程科学研究所中标广州港数字化管理系统，体现了水运科研院所在港口数字化建设中的技术支撑作用。

## 供应商格局分析

### 第一梯队：大型系统集成商

- **华海海图（北京）科技有限公司**：海事系统核心供应商，连续中标交通运输部海事局1988万元智慧海事综合管理系统和1735.8万元一体化建设项目，在海图显示与海事信息化领域占据主导地位。
- **葛洲坝通信技术有限公司**：港口方向大单供应商，中标江苏盐城港9950万元智慧港口安全能源管控项目。
- **浪潮软件集团有限公司**：中标福建海事局879万元VTS改造项目，在预警和监控系统领域有竞争力。
- **东华软件智能科技有限公司**：中标上海海图中心531万元S-101电子海图生产系统项目。

### 第二梯队：专业领域供应商

- **大连海大智龙科技有限公司**：模拟仿真和导航方向核心供应商，在多所院校中标航海模拟器、船舶操纵模拟器、船舶气象导航实验系统等项目。
- **大连海事大学**：作为高校参与产业化，中标275.78万元国产大型船舶操纵模拟器项目。
- **杭州麟云科技有限公司**：地方海事安全监测系统供应商，在四川多地中标水上安全监测项目。
- **天津路航信息科技有限公司**：电子航道图领域新兴供应商，中标河北省电子航道图测绘项目。
- **中电鸿信信息科技有限公司**：港口信息化供应商，中标119万元张家港港口信息化项目。

### 第三梯队：区域型供应商

- **厦门蓝色梦幻科技有限公司**：渔业方向，中标15.7万元数字化渔场建设项目。
- **厦门泓庚航海科技有限公司**：模拟器方向，中标集美大学船舶操纵模拟器升级改造项目。
- **海之韵（苏州）科技有限公司**：感知系统方向，中标70万元水路交通韧性全息感知系统。

## 市场驱动因素

### 政策驱动

1. **S-101标准升级**：IHO要求各成员国逐步从S-57向S-101标准迁移，推动海图生产系统全面升级。
2. **海洋牧场示范区建设**：国家级海洋牧场示范区评审推动信息化配套建设，遂溪、兴城、烟台等地项目均与此相关。
3. **智慧港口/航道规划**：交通运输部智慧港口和智慧航道建设规划持续释放市场需求。
4. **国产化替代**：教育领域设备更新和军队采购明确倾向国产化，推动大连海事大学、大连海大智龙等国产供应商发展。

### 技术驱动

1. **全息感知技术**：水路交通韧性全息感知系统等项目代表感知技术从单一传感器向多源融合演进。
2. **5G/物联网**：周口临港开发区等项目引入5G专网，支撑港口物联网应用。
3. **数字孪生**：航道通航仿真、港口数字化管理等项目体现数字孪生技术在港航领域的应用。
4. **大数据与AI**：海洋牧场大数据平台、智慧渔业监管决策平台等项目体现大数据和AI在渔业领域的应用。

## 投标策略建议

### 对海图/水域信息化企业

1. **把握S-101升级窗口**：S-101电子海图标准迁移是未来3-5年的确定性需求，具备S-101数据生产能力的供应商将获得先发优势。
2. **关注海洋牧场信息化**：沿海各省国家级海洋牧场示范区配套信息化项目密集释放，是增长最快的细分市场。
3. **布局智慧港口生态**：港口方向单体项目金额最大，但竞争也最激烈，需要与港口集团建立长期合作关系。

### 对模拟器/培训设备企业

1. **国产化替代机会**：航海模拟器国产化趋势明确，国内厂商应加大研发投入抢占市场。
2. **拓展应用场景**：从培训模拟向航道仿真、港口仿真等工程应用场景拓展。
3. **军队市场**：军队航海模拟仿真需求增长，但需要满足军品采购资质要求。

### 对渔业信息化企业

1. **区域深耕**：海洋牧场信息化项目具有明显的区域性，需要与地方农业农村局建立合作关系。
2. **平台化发展**：从单一监测向智慧渔业监管决策平台升级，提升项目附加值。
3. **关注内陆市场**：内陆智慧渔业项目开始出现，市场空间有待挖掘。

## 局限性说明

1. 本报告信息主要来源于公开招标网站搜索，部分项目金额和中标单位信息未公开，可能存在遗漏。
2. 军队采购项目的详细信息受限，无法获取完整数据。
3. 部分招标网站需要付费才能查看完整信息，本报告仅基于公开可获取的部分信息。
4. 市场容量估算基于已收集到的项目信息推算，实际市场规模可能更大。
5. 搜索时间有限，可能遗漏部分早期（2021-2022年）的项目信息。
