# S-52 Presentation Library Edition 4.0 检查指南

> 来源: [电子海图Presentation Library升级到Edition 4.0后相关注意事项](http://www.enclive.cn/News/NewsDetail.html?u=Trade&ID=32338)

## 一、版本背景

### 1.1 版本信息
- **当前版本**: IHO S-52 Presentation Library Edition 4.0
- **发布日期**: 2014年9月
- **实施日期**: 2017年6月16日经由海事安全大会通过实施
- **强制升级截止**: 2017年9月1日

### 1.2 版本变更说明
- PL 3.4及之前的检查方法不再适用
- 升级后通过查看 **ECDIS CHART 1** 来检查显示是否符合相关要求
- DPPC (Data Presentation and Performance Check) 概念引入

---

## 二、PL 4.0 升级后检查注意事项

### 2.1 船员检查准备事项

| 序号 | 检查项目 | 具体要求 |
|------|----------|----------|
| 1 | 设备升级证明 | 船员能够向检查官提供厂家出具的设备升级证明。如设备安装时就是PL4.0版本，需提供产品证书（证书上标明版本） |
| 2 | 版本显示 | 船员能够在设备中调取出当前版本，证明设备符合最新IHO ECDIS Presentation Library Edition 4.0要求 |
| 3 | ECDIS Chart 1 | 船员能够在设备中调出ECDIS Chart 1，并能够向检查官演示新符号的显示 |

### 2.2 证明ECDIS符合PL4.0的三种方法

#### 方法一：生产商声明
船舶进行PL4.0升级后，船上获得ECDIS生产商（如JRC、FURUNO等）出具的符合PL4.0的声明。

#### 方法二：海图信息读取
随机选择一张海图，进行海图信息的读取。例如JRC 901B机型：
- 在Chart List中选择一张海图
- 点击右键选择Information
- 查看Presentation Library版本号

#### 方法三：ECDIS Chart 1检查
通过ECDIS Chart 1检查新符号显示是否正常。

## 三、DPPC 检查体系

### 3.1 DPPC 定义
**DPPC** = Data Presentation and Performance Check（数据显示与性能检查）

- 用于检查ECDIS是否符合PL4.0标准
- 替代PL3.4之前的检查方法

### 3.2 Chart 1 检查数据
- Chart 1是用于执行ECDIS是否符合PL4.0标准检查的专用数据
- 设备升级到PL4.0后，原有DPPC检查数据不再适用
- 需使用新版Chart 1数据集进行检查

### 3.3 检查执行要求

#### 检查范围
- 船上每台使用ENC的ECDIS都应进行检查
- 除非确认ECDIS型号及软件版本相同

#### 检查准备
1. 检查数据（通常由ENC服务提供商通过CD-ROM或与ENC相同方式提供）
2. 记录检查结果的表格或文档

## 四、ECDIS Chart 1 使用指南

### 4.1 Chart 1 概述
ECDIS Chart 1 是包含ENC使用符号图例的数据集，应安装在所有型式认可的ECDIS上。

### 4.2 Chart 1 功能
- 验证ECDIS能够正确显示IHO S-52 Presentation Library Edition 4.0引入的新符号
- 作为符号参考和培训工具
- 检查设备符号化功能是否正常

### 4.3 操作步骤
1. 在ECDIS中打开ECDIS Chart 1数据集
2. 检查各类符号是否正确显示
3. 特别关注Ed 4.0新增符号

## 七、常见问题与解决方案

### 7.1 符号显示异常

| 问题现象 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 新符号不显示 | 软件版本未更新 | 联系厂家升级 |
| 颜色显示错误 | 颜色表未更新 | 重新加载颜色表 |
| Chart 1无法打开 | 数据集缺失 | 从服务提供商获取 |

### 7.2 检查不通过处理流程

1. 确认ECDIS软件版本
2. 检查Presentation Library版本号
3. 确认Chart 1数据集完整性
4. 联系设备厂家技术支持
5. 记录问题并报告公司
