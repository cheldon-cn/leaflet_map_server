## 2. 功能需求

### 2.1 Must Have（必须实现）

| ID | 需求描述 | 验收标准 | 优先级 | 来源 |
|----|----------|----------|--------|------|
| F001 | 水深预警功能：根据UKC计算结果触发不同级别的水深预警 | UKC<阈值时触发对应级别预警，预警信息包含位置、UKC值、建议措施 | P0 | 2.1节 |
| F002 | 气象预警功能：接收气象数据，匹配预警区域，生成气象预警 | 正确解析气象数据，匹配船舶位置与预警区域，生成包含风力、能见度等信息的预警 | P0 | 2.2节 |
| F003 | 碰撞风险预警功能：处理AIS数据，计算CPA/TCPA，评估碰撞风险 | CPA<TCPA阈值时触发预警，显示目标船舶信息、CPA/TCPA值、风险等级 | P0 | 2.3节 |
| F004 | 航道状态预警功能：监测航道关闭、航标异常、桥梁净高 | 及时获取航行通告，正确识别航道状态变化并触发预警 | P0 | 2.4节 |
| F005 | 偏航预警功能：监测船舶偏航距离，触发偏航预警 | 偏航距离超过阈值时触发对应级别预警 | P1 | 2.5.1节 |
| F006 | 超速预警功能：监测船舶航速，在限速区域触发超速预警 | 航速超过限制时触发对应级别预警 | P1 | 2.5.2节 |
| F007 | 禁航区预警功能：监测船舶与禁航区距离，触发禁航区预警 | 接近或进入禁航区时触发预警 | P1 | 2.5.3节 |
| F008 | 预警推送功能：根据预警级别和用户配置推送预警信息 | 按配置方式（应用内/声音/短信/邮件）正确推送，推送成功率>99% | P0 | 3节 |
| F009 | 预警配置管理功能：管理系统预设阈值和用户自定义配置 | 支持开启/关闭预警类型、调整阈值、选择推送方式、设置免打扰时段 | P1 | 4节 |
| F010 | 预警查询功能：提供预警列表、详情、历史查询 | 支持按类型、级别、时间筛选，响应时间<500ms | P1 | 7.2节 |
| F011 | 预警确认功能：处理用户预警确认和反馈 | 正确记录用户确认状态和反馈内容 | P1 | 7.3节 |

## 8. 附录

### 8.1 预警级别定义

| 级别 | 名称 | 颜色 | 说明 |
|------|------|------|------|
| 1 | 特别严重 | 红色 | 立即采取行动，生命安全威胁 |
| 2 | 严重 | 橙色 | 尽快采取行动，重大风险 |
| 3 | 较重 | 黄色 | 注意观察，潜在风险 |
| 4 | 一般 | 蓝色 | 提示信息，轻微风险 |

### 8.2 预警类型与阈值

| 预警类型 | 一级阈值 | 二级阈值 | 三级阈值 | 四级阈值 |
|----------|----------|----------|----------|----------|
| 水深预警(UKC) | <0.3m | <0.5m | <1.0m | <1.5m |
| 碰撞预警(CPA) | <0.5海里 | 0.5-1海里 | 1-2海里 | - |
| 碰撞预警(TCPA) | <6分钟 | 6-15分钟 | 15-30分钟 | - |
| 偏航预警 | >200m | 100-200m | 50-100m | - |
| 超速预警 | >150% | >120% | >110% | - |

### 8.3 推送策略

| 预警级别 | 推送方式 |
|----------|----------|
| 一级（红色） | 应用内+声音+短信 |
| 二级（橙色） | 应用内+声音 |
| 三级（黄色） | 应用内通知 |
| 四级（蓝色） | 应用内通知 |

### 8.4 术语说明

| 术语 | 说明 |
|------|------|
| UKC | Under Keel Clearance，富余水深 |
| CPA | Closest Point of Approach，最近会遇距离 |
| TCPA | Time to CPA，到达最近会遇点时间 |
| AIS | Automatic Identification System，自动识别系统 |
| VTS | Vessel Traffic Service，船舶交通服务 |
