## 五、发现详情

### 5.1 已解决问题详情

#### M1: 碰撞预警流程图CPA公式与代码不一致

**状态**: ✅ 已解决  
**解决版本**: v1.2  
**提出者**: 海事架构师  
**权重**: 0.83

**问题描述**:
碰撞预警流程图中的CPA计算公式 `CPA = |ΔR × sin(ΔC-ΔB)|` 与实际代码中的精确算法不一致，流程图过于简化。

**影响范围**:
- 文档可维护性降低
- 开发人员可能误解算法实现

**修改方法**:
流程图保留简化表示，在代码注释中明确说明"精确算法见7.2.2节"，并在补充设计细节中说明算法选择依据。

**验证方式**:
检查流程图与代码描述的一致性。

---
