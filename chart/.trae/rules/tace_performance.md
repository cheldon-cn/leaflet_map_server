# 性能优化策略

> **主规则**: @.trae/rules/task_continuous_execution.md

---

## 一、优化策略

| 策略 | 说明 | 效果 |
|------|------|------|
| 任务清单缓存 | 首次读取后缓存 | 减少IO |
| 增量状态更新 | 只更新变化部分 | 减少写入 |
| 批量操作 | 合并小任务 | 提高效率 |

## 三、性能指标定义

| 指标 | 定义 | 目标值 |
|------|------|--------|
| 单任务执行时间 | 从开始到完成的时间 | < 5分钟 |
| 状态检测时间 | 扫描任务清单的时间 | < 100ms |
| 内存使用峰值 | 执行过程中最大内存 | < 500MB |
| CPU使用率 | 平均CPU占用率 | < 80% |
| 任务成功率 | 成功任务数/总任务数 | > 95% |
