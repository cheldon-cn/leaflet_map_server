# 输出规范

> **主规则**: @.trae/rules/task_continuous_execution.md

---

## 一、任务执行输出

```markdown
## 任务执行完成

### ✅ 任务ID: T1 - 任务名称

**执行结果**:
- 状态: ✅ Done
- 工时: 实际Xh (预估Yh)
- 测试覆盖率: XX%

**下一步**: 继续执行T2...
```

---

## 二、里程碑进度报告

```markdown
## 里程碑完成报告

### 里程碑 M1: 基础框架

**进度统计**:
- 已完成任务: X/Y
- 完成百分比: XX%
```

---

## 三、最终完成报告

```markdown
## 任务清单执行完成报告

### 总体统计
- **总任务数**: X
- **已完成任务**: X ✅
- **总工时**: Xh
- **测试覆盖率**: XX%

### 性能统计
- **总执行时间**: X小时Y分钟
- **任务成功率**: XX%
```

---

## 四、性能监控指标

| 指标 | 目标值 |
|------|--------|
| 任务执行时间 | < 5分钟 |
| 状态检测时间 | < 100ms |
| 内存使用 | < 500MB |
| 任务成功率 | > 95% |
