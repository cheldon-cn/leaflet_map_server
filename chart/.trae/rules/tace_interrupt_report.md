# 中断报告格式

> **主规则**: @.trae/rules/task_continuous_execution.md

---

## 一、报告格式

```markdown
## 任务执行中断报告

**中断时间**: 2026-04-02 14:30:25
**中断原因**: 用户请求停止

### 执行进度
- 已完成任务: 15/48 (31.25%)
- 当前任务: T16 (进行中)
- 剩余任务: 33个

### 恢复执行
- "继续按任务清单执行"
- "重新执行任务清单"
```

---

## 二、进度文件格式

```json
{
  "execution": {
    "startTime": "2026-04-02T12:15:30Z",
    "stopTime": "2026-04-02T14:30:25Z",
    "stopReason": "user_request",
    "totalTasks": 48,
    "completedTasks": 15,
    "currentTask": "T16"
  }
}
```

---

## 三、中断类型

| 类型 | 说明 | 处理方式 |
|------|------|----------|
| 暂停 | 临时停止 | 保存进度，保持资源 |
| 停止 | 完全停止 | 保存进度，释放资源 |
| 强制停止 | 立即停止 | 不等待当前任务完成 |
