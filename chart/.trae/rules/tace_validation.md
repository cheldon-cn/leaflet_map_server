# 规则验证

> **主规则**: @.trae/rules/task_continuous_execution.md

---

## 一、执行前验证

| 验证项 | 说明 |
|--------|------|
| 任务清单文件存在 | 检查文件路径 |
| 任务ID唯一 | 检查重复ID |
| 依赖关系无循环 | 检查循环依赖 |
| 优先级正确（P0-P3） | 检查优先级范围 |
| 状态有效 | 检查状态值 |

---

## 二、执行中验证

| 验证项 | 说明 |
|--------|------|
| 按优先级执行 | 检查执行顺序 |
| 自动检测继续 | 检查持续执行 |
| 正确处理阻塞 | 检查异常处理 |
| 自动更新状态 | 检查状态更新 |

---

## 三、性能验证

| 指标 | 目标值 |
|------|--------|
| 任务执行时间 | < 5分钟 |
| 状态检测时间 | < 100ms |
| 内存使用 | < 500MB |
| 任务成功率 | > 95% |

---

## 四、失败条件

| 失败类型 | 说明 |
|----------|------|
| 自动终止 | 执行后未检测完成状态 |
| 需要多次触发 | 需用户多次发送"继续执行" |
| 优先级错误 | 未按优先级执行任务 |
