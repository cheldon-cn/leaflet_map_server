# 资源隔离机制

> **主规则**: @.trae/rules/task_continuous_execution.md

---

## 一、隔离维度

| 维度 | 说明 |
|------|------|
| 工作目录 | 每个任务独立工作空间 |
| 环境变量 | 任务特定环境配置 |
| 临时文件 | 独立临时目录 |
| 输出文件 | 独立输出目录 |

---

## 二、目录隔离

```
主工作目录: .trae/workspace/
任务工作目录: .trae/workspace/task_{id}_{timestamp}/
任务临时目录: .trae/temp/task_{id}/
任务输出目录: .trae/output/task_{id}/
```

---

## 三、环境变量隔离

| 变量 | 说明 |
|------|------|
| TASK_ID | 当前任务ID |
| TASK_WORKDIR | 任务工作目录 |
| TASK_TEMPDIR | 任务临时目录 |

---

## 四、清理机制

| 时机 | 操作 |
|------|------|
| 任务完成后 | 清理工作目录和临时目录 |
| 输出合并后 | 清理任务输出目录 |
| 执行终止后 | 清理所有任务目录 |

---

## 五、冲突检测

```markdown
IF 输出文件与主目录冲突 THEN
  → 重命名任务输出文件
  → 记录冲突日志
  → 合并到主目录
END IF
```
