# 执行前初始化

> **主规则**: @.trae/rules/task_continuous_execution.md

---

## 一、目录创建

| 目录 | 用途 |
|------|------|
| .trae/progress/ | 存储进度文件 |
| .trae/logs/ | 存储日志文件 |
| .trae/workspace/ | 任务工作空间 |

---

## 二、初始化流程

| 步骤 | 内容 | 验证 |
|------|------|------|
| Step 1 | 创建必要目录 | 检查存在性 |
| Step 2 | 验证目录写入权限 | 尝试写入测试文件 |
| Step 3 | 初始化进度文件 | 创建空文件 |
| Step 4 | 验证配置文件格式 | 检查JSON有效性 |
| Step 5 | 验证任务清单存在 | 检查文件路径 |

---

## 三、验证失败处理

```markdown
IF 目录创建失败 THEN
  → 输出错误: "无法创建目录 {path}"
  → 等待用户确认权限
END IF

IF 权限验证失败 THEN
  → 输出错误: "目录无写入权限"
  → 等待用户修复权限
END IF
```

---

## 四、初始化检查清单

```markdown
□ .trae/progress/ 目录已创建
□ .trae/logs/ 目录已创建
□ 目录写入权限验证通过
□ 配置文件格式正确
□ 任务清单文件存在
```
